## 🏆 The 6 ROSClaw-Native Standards (ABSOLUTE REQUIREMENTS)

### 1. Absolute Async (异步动作)
- EVERY physical action MUST use `async def`
- ALL ROS 2 action calls MUST use `await`
- NO synchronous blocking calls in the main path
- Pattern: `goal_handle = await self._action_client.send_goal_async(goal_msg)`

### 2. Flywheel-Ready JSON (飞轮就绪响应)
- ALL tool responses MUST return structured JSON
- REQUIRED fields:
  ```python
  {
    "status": "PENDING|EXECUTING|SUCCEEDED|FAILED|CANCELED",
    "action_id": "uuid-string",
    "action_type": "tool_name",
    "semantic_goal": "human-readable description",
    "timestamp_start": 1234567890.0,
    "timestamp_end": 1234567895.0,
    "duration_seconds": 5.0,
    "success": true|false,
    "error_code": "optional error code",
    "error_message": "optional error message",
    "safety": {
      "firewall_validated": true|false,
      "violations": []
    }
  }
  ```

### 3. Native Firewall Hook (防火墙集成)
- ANY tool that moves the robot PHYSICALLY MUST be decorated with `@mujoco_firewall`
- Pattern:
  ```python
  @mujoco_firewall(
      model_path="path/to/robot.xml",
      safety_level=SafetyLevel.STRICT|MODERATE|LENIENT
  )
  async def move_robot(...):
      ...
  ```
- Include BOTH firewalled and non-firewalled versions

### 4. Graceful Preemption (优雅抢占)
- MUST implement `cancel_action(task_id: str)` tool
- MUST track active tasks in `self._active_tasks: Dict[str, Any]`
- MUST handle `cancel_goal_async()` properly
- Pattern:
  ```python
  async def cancel_action(self, task_id: str) -> Dict:
      if task_id not in self._active_tasks:
          return {"success": False, "error": "No active action"}
      goal_handle = self._active_tasks[task_id]
      cancel_future = goal_handle.cancel_goal_async()
      cancel_result = await self._await_future(cancel_future)
      return {"success": True, "message": f"Action {task_id} canceled"}
  ```

### 5. State-Aware Affordance (状态感知)
- MUST implement local state machine
- MUST check state before executing actions
- MUST reject invalid operations
- Pattern:
  ```python
  if self.state.is_moving:
      return ActionResult(
          status=ActionStatus.REJECTED,
          error_message="Robot is already executing a motion. Cancel existing action first."
      )
  ```

### 6. Semantic Spatial Binding (语义空间绑定)
- MUST use TF2 for semantic frame lookups
- MUST provide `lookup_tf_pose(target_frame, reference_frame)` tool
- Pattern:
  ```python
  async def lookup_tf_pose(self, target_frame: str, reference_frame: str = "base_link"):
      transform = self._tf_buffer.lookup_transform(reference_frame, target_frame, rclpy.time.Time())
      return transform_to_pose(transform)
  ```
